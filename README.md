## 📁 Proyectos Destacados en Desarrollo

### 🔹 [Ecosistema DIRPOLES](https://github.com/tu-usuario/nombre-del-repositorio-central)
* **[DIRPOLES_4](https://github.com/tu-usuario/dirpoles_4):** Sistema core de gestión centralizada desarrollado en PHP 8 bajo arquitectura MVC orientada a objetos (PSR-4). Integra un modelo de **Base de Datos dual** diseñado específicamente para aislar de forma estricta las tablas críticas de seguridad (`dirpoles_security`) de los flujos ordinarios de negocio (`dirpoles_business`). Incorpora *Server-Sent Events (SSE)* para notificaciones en tiempo real, control riguroso de *CORS* y políticas de *Rate Limiting*.
* **[DIRPOLES_IA / Microservicio](https://github.com/tu-usuario/dirpoles_ia):** API REST independiente construida con Python y FastAPI. Este microservicio se conecta de forma aislada a la base de datos para extraer métricas complejas y consumir modelos de lenguaje avanzados (LLMs mediante Google AI Studio), automatizando por completo la generación de reportes analíticos y estadísticos estructurados de texto.
* **[DIRPOLES_APP](https://github.com/tu-usuario/dirpoles_app):** Extensión y aplicación móvil multiplataforma complementaria desarrollada con React Native y Expo, permitiendo el consumo seguro de las APIs del ecosistema directamente desde dispositivos móviles.
